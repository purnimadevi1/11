# Web-Development-Portfolio

This repository serves as a showcase of my skills and progress in front-end web development, primarily utilizing **HTML5** and **CSS3**. It includes various exercises, projects, and my personal portfolio page.

##  Key Projects & Features

This collection features several distinct projects and components:

| File Name | Description | Key Focus |
| :--- | :--- | :--- |
| **`Portfolio.html`** | My main personal portfolio page. | Layout, responsive design, contact details. |
| **`major.html`** | **[Update this]** A complex, multi-section web page for my major project. (e.g., A full e-commerce product page, a landing page for a startup, etc.) | Advanced CSS techniques (Flexbox/Grid), media queries. |
| **`mini.html`** | **[Update this]** A focused, smaller project or design challenge. (e.g., A pricing card component, a simple form, a photo gallery) | Specific CSS properties, basic structure. |
| **`week1.html`, `week2.html`** | Learning modules/exercises completed during initial stages of training. | Basic HTML structure, CSS selectors, box model. |
| **`*.css` files** | Dedicated style sheets for various pages. | Styling and visual design. |

## Technologies Used

* **HTML5:** For content structure and semantics.
* **CSS3:** For styling, layout, and visual presentation.
